## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MostafaTshahin/heba-raouf-bazaar.git
   cd heba-raouf-bazaar
   ```

2. **Open locally:**
   - Simply open `index.html` in your web browser

3. **Deploy to GitHub Pages:**
   - The site is automatically deployed at: `https://MostafaTshahin.github.io/heba-raouf-bazaar`

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #E1306C;      /* Instagram pink */
    --secondary-color: #000000;    /* Black */
    --accent-color: #1877F2;       /* Facebook blue */
    /* ... more colors ... */
}
```

### Content
- Edit `index.html` to update text, images, and links
- Modify social media URLs in the social buttons section

### Styling
- All styles are in `styles.css`
- Animations and responsive breakpoints are included
